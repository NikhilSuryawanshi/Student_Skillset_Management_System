Student Skill Set Management System

(SSSM)

Our mission:

In this world of growing technologies everything has been computerized. With large number of work opportunities the Human workforce has increased. Thus there is a need of a system which can handle the data of such a large number of Student’s skill information. This system simplifies the task of maintaining records because of well organised database. The objective of this project is to provide a comprehensive approach towards the management of student’s General and Skill information. This will be done by designing and implementing a General Students Skill Set Management System (SSSM) that will bring up a major paradigm shift in the way that student's Seneral and Skill information is handled.
